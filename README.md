# README

This is a rails application that assumes sqlite database.
It is an itinerary web application

Clone repo and then:

## development environment setup
```
bundle install
rails db:create:all
rails db:migrate
rails g rspec:install
```

## testing
```
rails g rspec:install
```
run suites:
```
rspec
```
